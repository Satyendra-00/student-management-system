# Student Management System

## Overview

The Student Management System is a simple React.js application developed to understand the basics of frontend development using React. The application allows users to add student details, display them dynamically, and delete records from the list.

## Features

* Add a new student
* Display a list of student records
* Delete a student record
* Dynamic UI updates using React State
* Reusable React Components

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS
* Vite

## Learning Outcomes

* Understanding React component structure
* Working with JSX syntax
* Using functional components
* Managing data with React State (`useState`)
* Passing data through Props
* Rendering lists dynamically using `map()`
* Handling user events

## Installation and Setup

1. Clone the repository:

```bash
git clone https://github.com/Satyendra-00/student-management-system.git
```

2. Navigate to the project folder:

```bash
cd student-management-system
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the application in your browser:

```text
http://localhost:5173
```
